# Legacy Cutover SQL

Diese Dateien gehoeren zum einmaligen Relaunch-Cutover vom alten Poll-Schema auf das aktuelle Match-/Appointment-Modell.

Sie sind bewusst archiviert und gehoeren nicht mehr zum aktiven Setup:

- lokale Neuinstallation: `supabase/schema/current.sql`
- optionaler manueller Bootstrap: `supabase/seeds/001_bootstrap_core_data.sql`
- lokale Beispieldaten: `scripts/generateLocalTestData.mjs`

Die SQL-Dateien hier nur noch als historische Referenz behalten, nicht erneut auf einer frischen DB ausfuehren.
